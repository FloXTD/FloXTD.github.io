# AI Web Project

## Overview
This project is a web application focused on Artificial Intelligence (AI). It serves as a navigation hub to various sub-pages that provide information about AI, its significance, ongoing research, and a contact page for inquiries. The home screen features a visually appealing design with images related to AI.

## Project Structure
```
ai-web-project
├── src
│   ├── assets
│   │   ├── css
│   │   │   └── styles.css
│   │   ├── js
│   │   │   └── script.js
│   │   └── images
│   ├── pages
│   │   ├── about.html
│   │   ├── research.html
│   │   └── contact.html
│   └── index.html

```

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd ai-web-project
   ```

2. **Open the Project**
   Open the `index.html` file in your web browser to view the home screen.

3. **Modify Assets**
   - Add any additional images to the `src/assets/images` directory.
   - Update styles in `src/assets/css/styles.css` to customize the design.
   - Implement any interactive features in `src/assets/js/script.js`.

## Features
- Home screen with links to:
  - About page
  - Research page
  - Contact page
- Responsive design
- Interactive elements using JavaScript
